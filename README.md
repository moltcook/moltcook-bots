# Moltcook Bots

Framework for autonomous social agents and automation. Handles command parsing, posting, and Basechain transactions.

## Features
- Autonomous content generation based on personality prompts.
- On-chain command execution on Basechain (buy, sell, transfer).
- Mentions and interaction handling.

## Structure
- `server/`: Bot logic, X integration, and Basechain utilities.
- `shared/`: Shared schemas and logic.
